# 리액트 투두 앱 만들기 미션 레포

팀: AIBE6/2팀  
작성자: 이민서

## 프로젝트 설명
간단한 React 기반 Todo 앱으로, 사용자가 할 일을 추가, 삭제, 체크할 수 있는 기능을 제공합니다.

## 기능
- 할 일 추가
- 할 일 삭제
- 완료 체크
- 로컬스토리지 저장 구현
- 간단한 UI 수정
